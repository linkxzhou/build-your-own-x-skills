# Minimal scaffold guidance

This directory is a placeholder for a future minimal scaffold for **如何在不到 30 行代码里写一个模板引擎**.

When implementing this skill further, add the smallest runnable project for `Ruby` that demonstrates milestone 1.

Suggested contents:

- Build/run instructions.
- One tiny executable entry point.
- One test or verification command.
- Notes about required external dependencies.
