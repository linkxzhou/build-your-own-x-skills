# Minimal scaffold guidance

This directory is a placeholder for a future minimal scaffold for **如何用不到 1000 行代码写一个视频播放器**.

When implementing this skill further, add the smallest runnable project for `C` that demonstrates milestone 1.

Suggested contents:

- Build/run instructions.
- One tiny executable entry point.
- One test or verification command.
- Notes about required external dependencies.
