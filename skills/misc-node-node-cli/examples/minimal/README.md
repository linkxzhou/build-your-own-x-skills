# Minimal scaffold guidance

This directory is a placeholder for a future minimal scaffold for **如何用 Node 创建一个真实世界的 Node CLI 应用**.

When implementing this skill further, add the smallest runnable project for `Node.js` that demonstrates milestone 1.

Suggested contents:

- Build/run instructions.
- One tiny executable entry point.
- One test or verification command.
- Notes about required external dependencies.
