# Minimal scaffold guidance

This directory is a placeholder for a future minimal scaffold for **从源代码到机器码：从零构建你自己的编译器**.

When implementing this skill further, add the smallest runnable project for `Python` that demonstrates milestone 1.

Suggested contents:

- Build/run instructions.
- One tiny executable entry point.
- One test or verification command.
- Notes about required external dependencies.
