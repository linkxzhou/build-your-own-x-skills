This directory is a placeholder for a future minimal scaffold for **45 步写数据库：一系列以 TDD 为主的小型编码谜题**.

When implementing this skill further, add the smallest runnable project for `Go` that demonstrates milestone 1.

- One tiny executable entry point.

- Notes about required external dependencies.
