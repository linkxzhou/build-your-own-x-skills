# Minimal scaffold guidance

This directory is a placeholder for a future minimal scaffold for **Beautiful Racket：如何用 Racket 制作自己的编程语言**.

When implementing this skill further, add the smallest runnable project for `Racket` that demonstrates milestone 1.

Suggested contents:

- Build/run instructions.
- One tiny executable entry point.
- One test or verification command.
- Notes about required external dependencies.
