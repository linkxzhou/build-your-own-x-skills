# Minimal scaffold guidance

This directory is a placeholder for a future minimal scaffold for **像程序员一样思考：只用 JS/HTML/CSS 做贪吃蛇**.

When implementing this skill further, add the smallest runnable project for `JavaScript` that demonstrates milestone 1.

Suggested contents:

- Build/run instructions.
- One tiny executable entry point.
- One test or verification command.
- Notes about required external dependencies.
