# 神经网络原理教学视频 - 场景设计文档

## 项目概述
- **主题**: 神经网络基础原理
- **目标受众**: 计算机专业学生、AI 初学者
- **预计时长**: 约 12-15 分钟
- **风格**: 教育科普，循序渐进

---

## 场景列表

### 场景 1: 开场引入 (~40秒)
**目标**: 引出神经网络的背景和重要性

**内容**:
1. 标题动画: "神经网络原理详解"
2. 提出问题: "计算机如何像人脑一样思考？"
3. 展示人脑神经元图示
4. 引出: "让我们从最基本的单元开始"

**视觉元素**:
- 大脑轮廓图
- 神经元连接动画
- 问号到灯泡的转换

---

### 场景 2: 生物神经元 (~60秒)
**目标**: 介绍生物神经元的结构和工作原理

**内容**:
1. 展示生物神经元结构
   - 树突 (Dendrites): 接收信号
   - 细胞体 (Cell Body): 处理信号
   - 轴突 (Axon): 传递信号
   - 突触 (Synapse): 连接其他神经元
2. 信号传递过程动画
3. 阈值激活概念

**视觉元素**:
- 神经元结构图
- 电信号流动动画
- 颜色: 蓝色=输入，绿色=处理，橙色=输出

---

### 场景 3: 感知机 - 最简单的神经网络 (~90秒)
**目标**: 介绍感知机模型

**内容**:
1. 从生物神经元到人工神经元的类比
2. 感知机结构:
   - 输入 x₁, x₂, ..., xₙ
   - 权重 w₁, w₂, ..., wₙ
   - 偏置 b
   - 激活函数
3. 数学公式: y = f(Σwᵢxᵢ + b)
4. 简单示例: AND 门的实现

**视觉元素**:
- 感知机结构图
- 权重连线（粗细表示权重大小）
- 公式动态推导

---

### 场景 4: 激活函数 (~90秒)
**目标**: 详解各种激活函数

**内容**:
1. 为什么需要激活函数？
   - 引入非线性
   - 否则多层网络等同于单层
2. 常见激活函数:
   - Step Function (阶跃函数)
   - Sigmoid: σ(x) = 1/(1+e⁻ˣ)
   - Tanh: tanh(x)
   - ReLU: max(0, x)
   - Leaky ReLU
3. 各函数图像对比
4. 选择建议

**视觉元素**:
- 函数图像绘制动画
- 导数图像对比
- 梯度消失问题可视化

---

### 场景 5: 多层神经网络结构 (~120秒)
**目标**: 展示完整的神经网络架构

**内容**:
1. 网络层次:
   - 输入层 (Input Layer)
   - 隐藏层 (Hidden Layers)
   - 输出层 (Output Layer)
2. 全连接网络结构
3. 信息前向传播过程
4. 数据流动动画演示

**视觉元素**:
- 多层网络结构图
- 节点和连接线
- 数据流动光效动画
- 颜色编码: 输入=蓝，隐藏=紫，输出=橙

---

### 场景 6: 前向传播详解 (~90秒)
**目标**: 详细演示前向传播计算过程

**内容**:
1. 单个神经元计算
2. 矩阵形式表示
3. 逐层计算示例
4. 完整数值演算

**数学公式**:
```
a⁽¹⁾ = f(W⁽¹⁾x + b⁽¹⁾)
a⁽²⁾ = f(W⁽²⁾a⁽¹⁾ + b⁽²⁾)
...
y = f(W⁽ⁿ⁾a⁽ⁿ⁻¹⁾ + b⁽ⁿ⁾)
```

**视觉元素**:
- 矩阵乘法动画
- 数值流动显示
- 激活函数应用

---

### 场景 7: 损失函数 (~60秒)
**目标**: 介绍损失函数的概念

**内容**:
1. 什么是损失函数？
   - 预测值与真实值的差距
2. 常见损失函数:
   - MSE: L = 1/n Σ(y - ŷ)²
   - Cross-Entropy: L = -Σylog(ŷ)
3. 损失曲面可视化
4. 优化目标: 最小化损失

**视觉元素**:
- 3D 损失曲面
- 误差可视化
- 预测 vs 真实对比

---

### 场景 8: 反向传播算法 (~150秒)
**目标**: 详解反向传播的核心原理

**内容**:
1. 链式法则回顾
2. 梯度的计算
3. 反向传播过程:
   - 从输出层开始
   - 逐层向后计算梯度
   - 更新权重
4. 数学推导 (简化版)
5. 动画演示梯度流动

**数学公式**:
```
∂L/∂w = ∂L/∂y · ∂y/∂z · ∂z/∂w
```

**视觉元素**:
- 梯度流动箭头（反向）
- 链式法则可视化
- 权重更新动画

---

### 场景 9: 梯度下降优化 (~90秒)
**目标**: 介绍梯度下降算法

**内容**:
1. 梯度下降直觉
   - 下山类比
2. 学习率的作用
3. 更新公式: w = w - α·∇L
4. 不同学习率效果对比
5. 优化器变种:
   - SGD
   - Momentum
   - Adam

**视觉元素**:
- 2D/3D 梯度下降动画
- 学习率对比图
- 收敛过程可视化

---

### 场景 10: 实际案例 - 手写数字识别 (~90秒)
**目标**: 展示神经网络的实际应用

**内容**:
1. MNIST 数据集介绍
2. 网络结构设计
   - 784 输入 (28×28)
   - 隐藏层
   - 10 输出 (0-9)
3. 训练过程可视化
4. 识别演示

**视觉元素**:
- 手写数字图片
- 网络处理过程
- 输出概率分布

---

### 场景 11: 过拟合与正则化 (~60秒)
**目标**: 介绍过拟合问题和解决方案

**内容**:
1. 什么是过拟合？
2. 训练集 vs 测试集曲线
3. 正则化方法:
   - L1/L2 正则化
   - Dropout
   - Early Stopping
4. 效果对比

**视觉元素**:
- 拟合曲线对比
- Dropout 可视化
- 损失曲线图

---

### 场景 12: 总结与展望 (~45秒)
**目标**: 回顾核心概念，展望未来

**内容**:
1. 核心概念回顾:
   - 神经元结构
   - 前向传播
   - 反向传播
   - 梯度下降
2. 神经网络的发展:
   - CNN, RNN, Transformer
3. 感谢观看

**视觉元素**:
- 知识点总结图
- 网络类型图标
- 结束动画
