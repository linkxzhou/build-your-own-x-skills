## Scene 2: 递推关系基础 (scene_02_recurrence_basics.py)

### 2.1 什么是递推关系
- 定义：用前面的项定义当前项
- 类比：数列接龙游戏
- 需要：初始值 + 递推规则

### 2.2 等差数列
- 规则：a_n = a_{n-1} + d
- 示例：1, 2, 3, 4, 5...（d=1）
- 动画：数列生长

### 2.3 等比数列
- 规则：a_n = r × a_{n-1}
- 示例：1, 2, 4, 8, 16...（r=2）
- 动画：指数增长可视化

### 2.4 斐波那契数列（重点）
- 规则：a_n = a_{n-1} + a_{n-2}
- 示例：1, 1, 2, 3, 5, 8, 13...
- 动画：兔子繁殖问题
- 黄金比例关联

### 3.2 伪随机数生成器

### 3.3 应用场景

## Scene 4: 非线性递推的奇妙世界 (scene_04_nonlinear_recurrence.py)

### 4.1 非线性递推特点
- 项之间做更复杂运算（乘法、平方等）
- 能产生更奇妙的现象

### 4.2 逻辑斯蒂映射（混沌入门）
- 公式：x_n = r × x_{n-1} × (1 - x_{n-1})
- 参数 r 变化时的行为：
  - r < 3：稳定点
  - 3 < r < 3.57：周期行为
  - r > 3.57：混沌
- 动画：分岔图生成
- 蝴蝶效应概念

### 4.3 曼德博集（选讲/简化）
- 公式：z_n = z_{n-1}² + c
- 复数平面上的迭代
- 判断是否"爆炸"
- 展示分形图案

## Scene 5: 解递推关系 (scene_05_solving_recurrence.py)

### 5.1 为什么需要直接公式
- 递推太慢，需要直接求第n项
- 引入特征方程方法

### 5.2 一阶递推的解法
- 示例：a_n = r × a_{n-1}
- 特征方程：x - r = 0
- 通解：a_n = A × r^n
- 用初始值确定 A

### 5.3 二阶递推的解法（斐波那契）
- 递推式：a_n = a_{n-1} + a_{n-2}
- 特征方程：x² - x - 1 = 0
- 两个根：黄金比例 φ 和 1-φ
- 通解：a_n = A × φ^n + B × (1-φ)^n
- 比内公式推导

### 5.4 比内公式的魔力
- 直接计算任意斐波那契数
- 动画：对比递推 vs 直接计算

## Scene 6: 递归基础概念 (scene_06_recursion_intro.py)

### 6.1 从递推到递归
- 递推：从前到后的生成
- 递归：从大到小的拆解与合并
- 函数调用自身

### 6.2 两个核心要素
- 基准情况（Base Case）：最简单的问题，直接解决
- 递归情况（Recursive Case）：拆分成更小的子问题
- 没有基准情况会怎样？无限循环！

### 6.3 趣味例子
- GNU = "GNU's Not Unix"（无限缩写）
- 乌龟塔故事
- 俄罗斯套娃动画

## Scene 7: 递归算法示例 (scene_07_recursion_examples.py)

### 7.1 阶乘
- 数学定义：n! = n × (n-1)!，1! = 1
- 代码展示
- 调用栈动画

### 7.2 乘方计算（快速幂）
- 朴素方法：乘 n 次
- 优化：b^n = (b^{n/2})²
- 问题规模减半
- 对比效率

### 7.3 斐波那契递归（警示）
- 直接递归代码
- 问题：大量重复计算
- 调用树展示
- 尾递归优化概念

### 7.4 快速排序
- 算法流程：
  1. 选基准值
  2. 小的放左边，大的放右边
  3. 递归处理左右子数组
- 分而治之策略
- 动画展示排序过程

### 7.5 汉诺塔
- 问题描述
- 递归分解：
  1. 移动 n-1 个盘子到 B
  2. 移动最大盘子到 C
  3. 移动 n-1 个盘子到 C
- 递推关系：M_n = 2 × M_{n-1} + 1
- 解：M_n = 2^n - 1
- 64层塔：世界末日的数学依据

## Scene 8: 总结与启示 (scene_08_summary.py)

### 8.1 递推与递归的关系
- 硬币的两面
- 递推：正向迭代构造
- 递归：逆向分解合并
- 核心思想：化繁为简，分步解决

### 8.2 效率的重要性
- 并非所有递归都高效
- 警惕重复计算
- 尾递归优化
- 递推（循环）的优势

### 8.3 编程启示
- 递归是强大的编程范式
- 适合树/图遍历、分治算法
- 函数式编程的基石
- 从数学到代码的桥梁

### 8.4 结语
- 掌握递推与递归 = 掌握"俄罗斯套娃"思维
- 通往更高级算法的关键一步
