# 布尔代数：真与假的数学

## 概述
- **主题**: 布尔代数——现代计算机科学的逻辑基石
- **Hook**: 为什么计算机只用0和1就能完成所有复杂运算？答案就在布尔代数中
- **目标受众**: 编程初学者、计算机科学入门者、对数字电路感兴趣的学习者
- **预计时长**: 约 15-18 分钟
- **核心洞见**: 布尔代数是连接抽象数学与物理电路的桥梁

## 叙事主线
从"开关数学"的直观理解开始，介绍布尔代数的三种基本运算和核心定律。然后展示布尔代数在集合论、命题逻辑、模2算术中的三个"分身"。接着从理论走向实践，讲解布尔函数和逻辑门。最后以构建二进制加法器为终极案例，展示布尔代数如何成为计算机的基石。

## Scene 3: 核心定律
**时长**: ~120 秒
**目的**: 介绍布尔代数的核心定律，建立公理系统概念

### 视觉元素
- 定律表格的动画展示
- 韦恩图验证
- 德摩根定律的特殊动画

### 内容
1. 交换律：a ∨ b = b ∨ a, a ∧ b = b ∧ a
2. 结合律：a ∨ (b ∨ c) = (a ∨ b) ∨ c
3. 分配律：a ∨ (b ∧ c) = (a ∨ b) ∧ (a ∨ c)
4. 同一律：a ∨ 0 = a, a ∧ 1 = a
5. 互补律：a ∨ ¬a = 1, a ∧ ¬a = 0
6. 德摩根定律：¬(a ∧ b) = ¬a ∨ ¬b

### 技术要点
- 使用卡片式布局展示定律
- 动画验证德摩根定律
- 强调对称性

## Scene 4: 三个分身实例
**时长**: ~150 秒
**目的**: 展示布尔代数在不同领域的具体实例

### 视觉元素
- 三列对比展示
- 集合论的韦恩图
- 命题逻辑的真值表
- 模2算术的计算演示

### 内容
1. **集合论**:
   - 幂集 P(U) 上的运算
   - ∩ = AND, ∪ = OR, 补集 = NOT
   - ∅ = 0, U = 1

2. **命题逻辑**:
   - {真, 假} 上的运算
   - 逻辑"且""或""非"
   - F = 0, T = 1
   - 程序中的 if/while

3. **模2整数算术 (ℤ₂)**:
   - {0, 1} 上的运算
   - 乘法 = AND
   - 加法 = XOR
   - 数字电路的直接模型

### 技术要点
- 三列并排对比
- 突出结构相同性
- 颜色一致性

### 内容

3. **NAND门的万能性**:

   - 用NAND构建AND
   - 用NAND构建OR
   - "一种门统治一切"

### 技术要点

## Scene 6: 二进制加法器
**时长**: ~150 秒
**目的**: 终极应用案例，展示如何用逻辑门构建加法器

### 视觉元素
- 半加器电路图
- 全加器电路图
- 8位加法器级联动画
- 进位传播动画

### 内容
1. **半加器 (Half Adder)**:
   - 输入：A, B
   - 输出：Sum (S), Carry (C)
   - S = A XOR B
   - C = A AND B
   - 电路图展示

2. **全加器 (Full Adder)**:
   - 输入：A, B, Cin
   - 输出：Sum, Cout
   - 由两个半加器和一个OR门组成
   - 电路图展示

3. **8位加法器**:
   - 8个全加器级联
   - 进位传播动画
   - 计算示例：01011001 + 00100111

### 技术要点
- 清晰的电路图布局
- 动画展示信号传播
- 颜色区分不同信号
