## Overview
- **Topic**: 深度学习核心原理可视化解析

2. **TPU简介**:

## Scene 4: 训练的艺术 - 前向传播与反向传播
**Duration**: ~150 seconds
**Purpose**: 理解深度学习的灵魂：前向传播、反向传播、梯度下降

### Visual Elements
- 多层流水线（层盒子）
- 特征图逐层抽象可视化
- 误差逆向回溯动画（红色细流）
- 参数微调旋转动画

### Content
1. **前向传播流水线**:
   - 猫图片（像素网格）从左侧进入
   - 经过第一个"层"盒子 → 边缘检测特征图
   - 继续流向下一层 → 更抽象特征（眼睛、鼻子部件）
   - 经过多个盒子后输出 `[0.9, 0.1]`（猫/狗概率）
   - 概率条动画显示"猫"柱子升高
   - "流水线的长度就是模型的深度"

2. **反向传播 - 学习发生的瞬间**:
   - 比较预测与真实标签，计算"误差"
   - 巨大红色"误差"数字出现（如"0.8"）
   - 红色数字分解成无数红色细流
   - 沿前向传播蓝色箭头**逆向回溯**
   - 每到一个层盒子，红流精准附着在参数上
   - 显示不同强度（梯度大小）

3. **梯度下降**:
   - 参数根据红流方向和强度开始微小旋转/移动
   - 公式: $w_{new} = w_{old} - \alpha \cdot \nabla L$
   - 再次前向传播，新误差"0.5"，变小了！
   - 反复迭代，模型越来越好

4. **深度的价值**:
   - 多层结构 → 学习极其复杂和抽象的特征层次
   - 底层：边缘、颜色
   - 中层：纹理、部件
   - 高层：整个物体甚至场景

### Narration Notes
- 反向传播是深度学习的基石
- 精确计算每个参数"应该承担多少责任"
- 深度带来强大的表达能力

### Technical Notes
- 流水线用 `RoundedRectangle` 盒子链
- 特征图用简化纹理图案
- 误差回传用红色粒子流 + `MoveAlongPath`
- 参数调整用 `Rotate` 或 `scale` 微动

## Scene 5: 损失函数与优化
**Duration**: ~100 seconds
**Purpose**: 深入理解损失函数类型和学习率影响

### Visual Elements
- MSE vs 交叉熵损失可视化
- 学习率对比动画（太小/太大/合适）
- 损失曲面上的三条下降路径

### Content
1. **损失函数类型**:
   - **MSE (均方误差)**: 用于预测数值
     - $L = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$
     - 预测房价、分数等连续值
   - **交叉熵**: 用于分类任务
     - $L = -\sum_{i=1}^{n}y_i\log(\hat{y}_i)$
     - 判断猫还是狗

2. **梯度下降直觉**:
   - 想象在全黑迷宫里找出口
   - 脚能感觉坡度（梯度）
   - 沿最陡下坡方向走一小步
   - 这一步的大小叫**学习率** α

3. **学习率的影响** (三条路径对比):
   - **α 太小**: 缓慢爬行，收敛太慢（蓝色路径）
   - **α 太大**: 来回震荡，可能发散（红色路径）
   - **α 合适**: 平稳收敛到最优（绿色路径）

4. 核心洞察：学习率是"最关键的超参数之一"

### Narration Notes
- MSE像测量距离，交叉熵像测量概率分布差异
- 学习率的选择是一门艺术

### Technical Notes
- 损失曲面用抛物面或等高线
- 三条路径用不同颜色的 `Dot` 移动轨迹
- 使用 `ValueTracker` 控制动画速度差异

## Scene 6: 梯度挑战与深度的秘密
**Duration**: ~120 seconds
**Purpose**: 理解链式法则、梯度消失/爆炸，以及深度网络的表示学习能力

### Visual Elements
- 链式法则可视化（乘积链）
- 梯度消失动画（信号衰减）
- 数据点分类渐变变形动画（核心亮点）

### Content
1. **反向传播的数学 - 链式法则**:
   - 从输出的错误倒推每个参数的责任
   - $\frac{\partial L}{\partial w} = \frac{\partial L}{\partial y} \cdot \frac{\partial y}{\partial z} \cdot \frac{\partial z}{\partial w}$
   - 让海量参数的高效调整成为可能

2. **梯度消失/爆炸问题**:
   - 梯度经过很多层，连乘变得极小或极大
   - 前面层收不到有效信号，学不到东西
   - 解决思路：把模型设计得更容易优化（跳跃连接、归一化等）

3. **深度的价值 - 核心动画**:
   - 展示8层小网络分类两类点
   - 开始：两类点（红、蓝）混在一起，无法用直线分开
   - 每层对数据进行微小"扭曲"变换
   - Layer 1 → Layer 2 → ... → Layer 8
   - 最终：两类点被拉开，可以用一条直线分开
   - 最后一层只需"画条线"即可分类

4. 核心洞察：通过一系列变换，把杂乱数据变成易处理的形式

### Narration Notes
- 深度网络学习的是"数据的表示"
- 每一层都在"重新编码"数据
- 这就是"表示学习"的威力

### Technical Notes
- 链式法则用箭头连接的偏导数块
- 梯度消失用颜色渐淡的梯度流
- 数据点变形用 `always_redraw` + 插值变换
- 分割线最后 `Create` 出现

## Scene 7: 模型组件库 - 构建AI大脑的乐高积木
**Duration**: ~180 seconds
**Purpose**: 认识组成深度网络的基本组件

### Visual Elements
- 各种层的标准化可视化块
- 卷积滑动窗口动画
- 注意力权重连线动画
- 跳跃连接"高速公路"

### Content
1. **引入**: 神经网络由标准化"层"堆叠而成，像乐高积木

2. **线性层 (全连接层)**:
   - 最基础组件：$y = Wx + b$
   - 每个输入与每个输出全连接
   - W是权重矩阵，b是偏置

3. **激活函数**:
   - 引入非线性，是神经网络拟合复杂函数的关键
   - **ReLU**: $\text{ReLU}(x) = \max(0, x)$
   - 负数变0，正数保留（阈值过滤器）
   - 还有Sigmoid、Tanh等

4. **卷积层** (CV基石):
   - 3×3卷积核在特征图上滑动
   - 每个位置进行点乘求和
   - **参数共享**: 同一滤波器扫描整图，大大减少参数
   - **平移等变性**: 猫在左边或右边都能识别

5. **池化层**:
   - 下采样，减少空间尺寸
   - 最大池化：区域内只保留最大值
   - 保留主要信息，增强鲁棒性

6. **Dropout**:
   - 训练时随机"丢弃"部分神经元（置零）
   - 强迫网络不依赖任何单个神经元
   - 提高泛化能力，防止过拟合

7. **批量归一化 (BatchNorm)**:
   - 深度网络的"稳定器"
   - 强制每层输入分布稳定（零均值、单位方差）
   - 显著加速训练，缓解梯度问题

8. **跳跃连接 (Skip Connection)**:
   - 数据从某层直接"跳过"中间层，加到后面
   - 给梯度开"绿色通道"
   - 解决极深网络梯度消失难题
   - ResNet的核心思想

9. **注意力层** (核心动画):
   - 场景：单词序列 `["The", "cat", "sat", "on", "the", "mat"]`
   - 处理"sat"时，发射"查询"光束到其他单词
   - 每个单词回馈"键-值"对
   - 光束强度（注意力权重）= 查询与键的相似度
   - "sat"对"cat"和"on"注意力最强
   - 加权汇总形成新的上下文表示
   - $\text{Attention}(Q, K, V) = \text{softmax}(\frac{QK^T}{\sqrt{d_k}})V$

### Narration Notes
- 每个组件解决特定问题
- 卷积解决图像的局部特征提取
- 注意力解决长距离依赖
- 跳跃连接解决深度训练

### Technical Notes
- 每种层用不同颜色和形状的块
- 卷积用滑动窗口 + 高亮
- 注意力用连线 + 权重粗细变化
- 跳跃连接用弧形绕过箭头

## Scene 8: 经典架构 - 组装乐高，成就杰作
**Duration**: ~150 seconds
**Purpose**: 认识MLP、CNN/ResNet、Transformer三大架构家族

### Visual Elements
- MLP层叠结构图
- CNN卷积-池化-全连接流程
- ResNet残差块详细结构
- Transformer编码器-解码器架构

### Content
1. **多层感知机 (MLP)**:
   - 最原始但最能体现原理的深度网络
   - 多个"全连接层+激活函数"简单堆叠
   - 证明了"深度+非线性"的威力

2. **卷积神经网络 (CNN)** - 图像之王:
   - **LeNet-5 (1998)**: 首次成功用于手写数字识别
     - 验证了"卷积+池化"范式
   - **ResNet (2015)**: 里程碑式突破
     - 通过跳跃连接将网络做到100+层
     - 横扫图像分类比赛
     - 残差块：学习 $F(x) + x$ 而非直接学习 $H(x)$
   - **U-Net, DeepLab**: 编码器-解码器结构
     - 用于图像分割（像素级分类）

3. **Transformer** - 统治性架构:
   - 核心：自注意力机制 + 前馈网络
   - 配合跳跃连接和层归一化
   - 高效捕捉序列中任意长距离依赖
   - **原始Transformer**: 编码器+解码器，用于机器翻译
   - **GPT系列**: 基于解码器的自回归模型
     - 只用上文预测下一个词
     - 海量文本训练获得惊人能力
   - **Vision Transformer (ViT)**: 大胆创新
     - 图像切割成块，像单词序列处理
     - 纯Transformer处理图像，效果匹敌甚至超越CNN

### Narration Notes
- CNN专为图像设计，利用空间局部性
- ResNet证明了"更深≠更难训练"
- Transformer从NLP横扫到多模态

### Technical Notes
- 网络结构用简化方块图
- ResNet残差块展示 $x + F(x)$ 结构
- Transformer用多头注意力连线可视化

### Equations to Render

2. `\text{ReLU}(x) = \max(0, x)` - 激活函数 (Scene 7)

4. `L = -\sum_{i=1}^{n}y_i\log(\hat{y}_i)` - 交叉熵损失 (Scene 5)
5. `w_{new} = w_{old} - \alpha \cdot \nabla L` - 梯度下降 (Scene 4, 5)
6. `\frac{\partial L}{\partial w} = \frac{\partial L}{\partial y} \cdot \frac{\partial y}{\partial z} \cdot \frac{\partial z}{\partial w}` - 链式法则 (Scene 6)
7. `\text{Attention}(Q, K, V) = \text{softmax}(\frac{QK^T}{\sqrt{d_k}})V` - 自注意力 (Scene 7)
8. `H(x) = F(x) + x` - 残差连接 (Scene 8)
