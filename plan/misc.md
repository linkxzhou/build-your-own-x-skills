# 构建你自己的 未分类：skill 转换计划

- **分类 slug**：`misc`
- **项目数量**：62
- **目标平台**：Claude Code skills（同时兼容本仓库 `skills/` 目录约定）

## 分类级转换策略

将 `未分类` 分类下的每个教程转换为面向实践学习的开发技能。优先保持“一教程一 skill”，后续可按语言或共同核心抽象合并。
